## Описание 
Приложение является ReactJS проектом. Основные компоненты реализованы на языке TypeScript.
### `npm run start`
Запускает приложение на 3006 порту.
### `json-server tasks.json`
Запускает сервер с множеством задач на 3000 порту. Файл с данными **tasks.json** находиться в корне проектся 
### Структура компонентов /src/components/
1. **App** - основной компонент приложения. Содержит в себе компонент **container**
2. **container** - компонент отвечающий за работу с сервером. Работа с сервером происходит в след. методах: 
* **getAllTasks** - GET запрос на получение списка всех задач.
* **addNewTask** - POST запрос на добавление нового задачи.
* **updateComplitingTask** - PATCH запрос на обновление статуса выполнения задачи
* **deleteTask** - DELETE запрос на удаление задачи
* **updateTaskName** - PATCH запрос на изменение имени задачи
Container содержит в себе массив задач в state и компонент list.
3. **list** - компонент содержащий в себе текстовое поле, кнопку "ADD", и множество компонентов **taskBlock**
4. **taskBlock** - компонент задачи. Содержит в себе checkBox, текстовое поле с названием и кнопка-крестик для удаления <br /> Изменение имени задачи привязано к событию потери фокуса у текстового поля. То есть для того что бы изменить имя, необходимо нажать на название задачи, внести изменения, и нажать а любое свободно место на экране
### Модели /src/models/
В рамках данной системы используется только одна модель: **Task** <br />
Модель содержит в себе поля: 
* **id: string** - идентификатор задачи. Используется UUID формат для идентификаторов. 
* **text: string** - текст задачи.
* **isComplete: boolean** - состояние задачи, выполнена или нет.
